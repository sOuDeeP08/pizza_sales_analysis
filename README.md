# pizza_sales_analysis

📌 Overview

This project focuses on analyzing pizza sales data to uncover key business insights related to revenue, customer behavior, and product performance using SQL.

🎯 Objectives
Analyze total orders and revenue
Identify top-performing pizzas
Understand customer preferences
Generate actionable business insights

🗂️ Dataset
The database consists of the following tables:
orders → order_id, order_date, order_time
order_details → order_details_id, order_id, pizza_id, quantity
pizzas → pizza_id, pizza_type_id, size, price
pizza_types → pizza_type_id, name, category

🛠️ Tools & Technologies
SQL (MySQL)
Data Analysis

📊 Key Insights
Total orders exceeded 21,000+, indicating strong demand
Revenue generated was approximately $800K+
A small group of pizzas contributed to the majority of sales (Pareto trend)
Large-size pizzas were the most preferred by customers
Top-selling items significantly influenced overall revenue

💡 Business Recommendations
Focus marketing on top-performing pizzas
Optimize inventory for high-demand items
Promote premium pizzas to increase margins
Use data insights for pricing and bundling strategies

🚀 Conclusion
This project demonstrates how SQL can be used to transform raw data into meaningful business insights and support data-driven decision-making.
